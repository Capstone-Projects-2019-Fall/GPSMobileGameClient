Detailed Description

Location contains orientation, latitude, longitute, accuracy and timestamp.

This is what will contain all the curernt location information.

It also tracks if device has location services enabled and if device is using GPS or a network. Here is the Mapbox API link for location: https://docs.mapbox.com/mapbox-unity-sdk/api/unity/Mapbox.Unity.Location.Location.html
