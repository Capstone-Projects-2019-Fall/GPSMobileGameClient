Detailed Description

The EditorLocationProvider is responsible for providing mock location and heading data for testing purposes in the Unity editor.

This will allow us to test locations by not actually physically being at that location within Unity. For more information: https://docs.mapbox.com/mapbox-unity-sdk/api/unity/Mapbox.Unity.Map.MapVisualizer.html
